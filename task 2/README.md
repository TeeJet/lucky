Описание
------------
Определение максимального количества онлайн

Установка при помощи Docker
------------
Запустите контейнеры и сервер из корневой папки проекта
```
docker-compose up -d
```
Выполните миграции
```
docker exec -ti task2_php_1 ./yii migrate
```

Использование
-----
Запустите сервис на странице:
```
http://localhost:8003
```
